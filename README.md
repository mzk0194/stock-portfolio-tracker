# stock-portfolio-tracker

A simple stock portfolio tracker built with vanilla JavaScript.
This project was developed step by step from basic DOM manipulation to API integration and profit calculation logic.

Level 1–3

Add stock name, shares, and price

Render stock list dynamically //HTML에 미리 써둔게 아니라 js가 실행될때마다 새로 화면에 띄움(배열 데이터를 화면으로 변환)

Delete stocks

Calculate total value

Level 4–7

LocalStorage integration

Data persistence after refresh //새로고침해도 데이터가 유지됨

Code structure improvement

Level 8

Real-time stock price fetching using Alpha Vantage API

Async / Await usage

Error handling for API limits

Level 9

Separate buy price and current price

Individual stock profit calculation

Total portfolio profit calculation

Portfolio return percentage (%)

Dynamic color styling (green/red for profit/loss)

Automatic price update using setInterval
